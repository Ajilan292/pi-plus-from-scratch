# PI Plus From Scratch (ROS1)

> 目标：让你从 0 到 1 跑通 **上电 → 联网 → ROS1 连通 → 启动主 launch → RViz 验收 → Teleop**。

## 3 分钟 Quickstart
1. Setup → First Boot Checklist（首次跑通验收）
2. Setup → ROS1 Discovery（多机 / ROS_MASTER_URI / ROS_IP）
3. Software → Launch Matrix（按机型/配置选启动组合）
4. Troubleshooting → Diagnostics（启动失败先看这里）

## 我是谁？
- 教学/学生：看 Tutorials（每页都有验收标准）
- 开发者：看 Software（Packages / ROS API / Config System）
- 运维/集成：看 Setup + Troubleshooting（网络、多机、排障）

## 常用入口
- 首次跑通：Setup / First Boot Checklist
- ROS1 多机：Setup / ROS1 Discovery
- 启动组合：Software / Launch Matrix
- ROS 接口表：Software / ROS API Reference
- 排障入口：Troubleshooting / Diagnostics
